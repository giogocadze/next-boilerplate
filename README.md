# Next.js Boilerplate

Minimal Next.js boilerplate with a clean setup.

## Included
- Next.js (App Router) + TypeScript
- Tailwind CSS
- ESLint + Prettier
- Husky + lint-staged (pre-commit)

## Setup
```bash
npm install
npm run dev
```
## Scripts
```
npm run dev
npm run build
npm run start
npm run lint
npm run format
```
